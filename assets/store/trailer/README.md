# Trailer Notes

The final trailer `.mp4` is uploaded directly to the Steamworks partner site and is not committed to this repository.

## Source

- Capture gameplay clips using OBS at 1920×1080, 60 fps.
- Recommended clips (30s each): one per spell archetype showcasing the spell in combat + 30s boss montage.
- Ensure HUD is visible, no debug overlays, no smoke result panel.

## Edit Target

- **Length:** 60–90 seconds.
- **Structure:** Title card → gameplay montage → capsule art callout → "Wishlist on Steam" end card.
- Export as H.264 `.mp4`. See Steamworks documentation for accepted codecs and maximum file size.

## Working Directory

If editing locally, place source clips and project files in `assets/store/trailer/working/`. This directory is gitignored.

## Upload

Upload the final `.mp4` via the Steamworks partner site under Store Page → Trailer. Do not commit the binary to this repository.
